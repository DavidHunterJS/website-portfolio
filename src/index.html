Projects

Bathroom Chore Calendar (BCC)


The BCC app is a way to easily assign bathroom chore clean-up duties equally amongst a group of people.

There are thee bathrooms that need to be cleaned every day. This app will create a monthly calendar with three names on each day that correspond to a bathroom that they're assigned to.

This was built using CoffeeScript, FullCalendar and jQueryUI.
